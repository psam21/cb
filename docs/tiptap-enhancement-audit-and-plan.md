## 4. Technical Considerations

### üîÑ **Markdown Compatibility Matrix**

| Feature | Markdown Support | Notes |
|---------|-----------------|-------|
| Bold/Italic | ‚úÖ Full | `**bold**` `*italic*` |
| Headings | ‚úÖ Full | `# H1` `## H2` |
| Lists | ‚úÖ Full | `- item` `1. item` |
| Links | ‚úÖ Full | `[text](url)` |
| Images | ‚úÖ Full | `![alt](url)` |
| Code/Code Block | ‚úÖ Full | `` `code` `` or ` ```lang` |
| Blockquote | ‚úÖ Full | `> quote` |
| HR | ‚úÖ Full | `---` or `***` |
| Strikethrough | ‚úÖ With GFM | `~~text~~` |
| Tables | ‚úÖ With GFM | Pipe tables |
| Task Lists | ‚úÖ With GFM | `- [ ] task` |
| Underline | ‚ùå No standard | May lose on export |
| Text Color | ‚ùå None | Will lose on export |
| Highlight | ‚ùå None | Will lose on export |
| Font Size/Family | ‚ùå None | Will lose on export |
| Text Align | ‚ùå None | Will lose on export |
| Video Embeds | ‚ö†Ô∏è Partial | Can use plain URLs |
| Subscript/Superscript | ‚ö†Ô∏è Partial | Some parsers support |

### üéØ **Nostr NIP-23 Compatibility**

**Safe Features (Nostr-friendly):**
- ‚úÖ All basic markdown (bold, italic, headings, lists, links)
- ‚úÖ Code blocks
- ‚úÖ Blockquotes
- ‚úÖ Images
- ‚úÖ Strikethrough (with GFM)
- ‚úÖ Tables (with GFM)
- ‚úÖ Task lists (with GFM)

**Lossy Features (HTML-only):**
- ‚ùå Text colors/highlights
- ‚ùå Font changes
- ‚ùå Text alignment
- ‚ùå Underline (might work with HTML in some clients)

**Recommendation:** Add visual warning when using non-markdown features
